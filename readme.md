# Social App Platform

A full stack social application under active development.  
This project is built with a scalable architecture using Golang for backend services and TypeScript with Vite for the frontend. It focuses on robust authentication secure media handling and long term extensibility.

## Project Overview

The system is split into three main parts

1 Auth Backend API in Golang  
2 Frontend Web App in TypeScript using Vite  
3 CDN Backend API in Golang for media storage and delivery  

The goal is to build a production ready social platform with strong security clear separation of concerns and support for ongoing feature expansion.

## Tech Stack

### Backend Auth API

- Language Go
- OTP and password reset support
- Role based access control
- PostgreSQL or MySQL

### CDN Backend API

- Language Go
- Secure media upload endpoints
- File type and size validation
- Structured storage mapping

### Frontend

- TypeScript
- Vite
- Modern component based UI i.e Shadcn UI
- Auth state management
- API driven architecture
- Designed for mobile and desktop responsiveness
- Yet Simple Approach
